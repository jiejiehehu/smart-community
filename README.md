# 智慧社区物业管理平台 - 高保真演示原型

基于 Flask + SQLite + 原生 HTML/CSS/JS 的智慧社区物业管理平台，已升级为客户演示级高保真原型。

## 快速启动

```bash
cd 家政服务
bash start.sh
```

然后在浏览器访问：
- **PC管理端**: http://localhost:8888
- **H5居民端**: http://localhost:8888/h5.html

## 演示流程

### PC管理端演示流程
1. 首次打开显示演示引导弹窗
2. 点击"开始演示"进入系统
3. 右侧底部"❓"按钮可打开功能导航
4. 点击右上角"📱 居民端预览"切换到H5居民端

### 核心演示路径
1. **数据看板** → 查看社区运营总览
2. **家政服务** → 体验服务预约流程
3. **食堂管理** → 查看菜品和订单管理
4. **物业缴费** → 账单查询和在线缴费
5. **报修管理** → 工单提交和处理
6. **康养服务** → 健康监测和SOS紧急呼叫
7. **安防监控** → 门禁和告警管理

### H5居民端演示流程
1. 首次打开显示欢迎引导
2. 首页展示Banner、快捷入口、公告、订单
3. 底部Tab切换：首页/服务/消息/我的
4. 服务页可体验各类预约功能
5. 点击右上角"🖥️ 管理后台"切换到PC端

## H5居民端功能

### 页面结构
- **首页Tab**: Banner轮播、快捷功能(8个)、社区公告、最近订单
- **服务Tab**: 服务分类列表，支持家政预约、食堂订餐、物业缴费、报修申请等
- **消息Tab**: 消息分类（缴费提醒/报修进度/活动通知/系统消息）
- **我的Tab**: 用户信息、我的订单/缴费/报修、家庭成员、设置

### 关键详情页
- 🧹 **家政预约**: 服务类型选择→日期时间→联系方式→提交
- 🍽️ **食堂订餐**: 今日菜品列表→加入购物车→确认订单
- ❤️ **康养服务**: 血压/血糖曲线图 + SOS大按钮
- 🔧 **报修申请**: 类型选择→拍照上传→描述→提交
- 💰 **物业缴费**: 账单列表→立即缴费
- 🚗 **车位查询**: 我的车位信息展示
- 👥 **访客邀请**: 访客信息→发送邀请

## 技术架构

```
前端（PC端）: 原生 HTML + CSS + JavaScript
├── 扁平化设计风格
├── Fetch API 调用后端
├── 响应式布局
└── CSS动画效果

前端（H5端）: 原生 HTML + CSS + JavaScript
├── 移动端优先设计
├── 纯前端静态演示（无需后端）
└── 触摸友好交互

后端: Python Flask
├── RESTful API 设计
├── SQLite 轻量数据库
└── CORS 跨域支持
```

## 文件结构


## API 接口

- `/api/stats` - 全局统计
- `/api/housekeeping/*` - 家政服务
- `/api/canteen/*` - 食堂管理
- `/api/property/*` - 物业缴费
- `/api/repair/*` - 报修管理
- `/api/visitor/*` - 访客管理
- `/api/announcements` - 公告通知
- `/api/parking/*` - 车位管理
- `/api/health/*` - 健康监测
- `/api/care/*` - 照护计划
- `/api/sos/*` - 紧急呼叫
- `/api/activities` - 社区活动
- `/api/security/*` - 安防监控
