# 智慧社区物业管理平台 - 需求设计规格说明书

## 1. 项目概述

### 1.1 项目名称
智慧社区物业管理平台

### 1.2 项目类型
高保真Web演示原型 - 基于Flask+SQLite的前后端分离架构

### 1.3 核心功能概述
面向物业管理的数字化运营平台，提供PC管理端和H5居民端两种访问入口，覆盖11个核心业务模块。

### 1.4 目标用户
| 用户角色 | 使用场景 |
|---------|---------|
| 物业管理人员 | PC端进行日常运营管理、数据统计、订单处理 |
| 社区居民 | H5端进行服务预约、缴费、报修等自助服务 |

### 1.5 技术栈
- **后端**: Python Flask + SQLite
- **前端**: 原生 HTML + CSS + JavaScript（无框架依赖）
- **部署**: 支持本地运行和 Render 云平台部署
