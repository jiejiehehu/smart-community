### 踩坑经验

1. **Edit 工具匹配问题**：当文件中有多个相似字符串时，Edit 工具可能匹配到错误位置，导致修改不生效。解决方案：使用 Python 脚本进行精确替换，或直接使用 Write 工具重写整个文件。
2. **模板字面量语法错误**：在对象中使用模板字面量时，如果模板字符串包含特殊字符（如 `$`、反引号），可能导致解析错误。解决方案：改用字符串拼接或 `if/else if` 结构。
3. **HTML 中 CSS 泄漏**：如果 `<style>` 标签被过早闭合，后面的 CSS 代码会被浏览器当作文本内容显示。解决方案：检查 `<style>` 和 `</style>` 的配对情况。
